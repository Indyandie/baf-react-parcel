# baf-react-parcel ⚛️ 📦
Super basic react template using parcel.

## Excute it 
```bash
npx baf-react-parcel
```

---

## Install package globally
In your terminal

```bash
npm i -g baf-react-parcel
```

## Create baf-react-parcel
Run
```bash
   barp # will clone and intall in /barp if it doesn't already exist
```

Use custom name
```bash
   barp <dir-name>
```
